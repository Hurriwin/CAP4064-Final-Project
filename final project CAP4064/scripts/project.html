Capstone Project

For my Capstone Project my group and I decided to create an app for the school that would help the students and the professors to schedule after class meetings with one another.

We decided to create the application for both PC and mobile users.

For my portion of the project I worked as the database manager. My role was to implement the database for the application.

In total we had 6 members working on the project including myself.

Key fetures

  • Allowed direct and quick contact with professors
  • App was available on PC and mobile devices
  • Different indivual features for students and professors
  • User interface was simple and easy to use

Database Management

For the final project in my database management class my team had to create our own database. We chose to create our database on a daycare center.

Content in database

  • Names of kids and parents
  • Phone numbers of parents
  • Age and names of both kids and parents
  • Information on the staff of the daycare such as names, phone numbers, etc...

For this project we had to make sure that all the information we put in our database was non redundant, meaning no repeating information and the information had to be unique.

There was 3 members in my group including myself.

We also had to make our database work with c++ meaning that we could give the prompts commands and it will execute them. For example, if we wanted to find the name of a parent using their phone number we could enter the command to do so and it will return the name of that parent.

C++ Coding Project

For the final project of fundamentals of programming my partner and I had to make a progam in c++ that would recieve x and y cordinates from the user and mark the location with an "X" on a grid plane

Key Features

  • Could recall the location of previous locations given to the program
  • Can move the "X" by a certain amount of units when you give it the input
  • Can save current location to use later on
  • Has an undo feature to revert to a certain position

My and my partner created the program only using c++ and was a good project to utilize everything we learned in that class.
